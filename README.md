#### 介绍
springboot2.1.2集成quartz，动态管理job

-----------------------------------------------------------------------------------------------

#### 使用框架

1. 核心框架: SpringBoot 2.1.2
2. 持久层框架: mybatis-plus 3.0.7
3. 数据库：mysql 8.0.12

#### 使用说明

1. 修改application.yml里的数据库名称，导入sql文件(src/test/resources/sql/schedule_job.sql) 
2. 启动DemoApplication.java
